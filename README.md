# Customer Churn Prediction

## Project Overview
This project predicts whether a customer is likely to churn using Machine Learning.

## Dataset
Customer churn dataset containing customer details like:
- Age
- Gender
- Tenure
- Monthly Charges
- Contract Type
- Internet Service
- Payment Method

## Machine Learning Model
Algorithm used:
- Random Forest Classifier
